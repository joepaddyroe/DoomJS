DoomJS net demo

Two-browser Quantum-style input demo (not full Doom). Start DoomJSRelay, open this page in two tabs: one creates a room, the other joins. Both run the same deterministic ECS sim; checksums should match.

Serve from the DoomJS folder (the one with index.html), not DoomJSRelay: python -m http.server 8080 then open http://127.0.0.1:8080/demo/net-demo.html. Relay separately: cd DoomJSRelay && npm start.
